This program runs off of Node.js ensure that you have Node installed on your computer

To run this program: 

- Navigate to this directory with your console
- Type in "npm install" to install node module packages needed for this application
- When its done, type in "node app.js" to run the application
- Open your browser and open "http://localhost:3000/index" to get to the main testing page

Note:

- app.js is the node/hosting application that is used for endpoints and writing to the log file
- index.html is the codebase for the testing framework and collects and passes data to passwordShape.html to verify the password and go to the next steps
- passwordShape.html is the codebase for the password creation and entry for the password scheme itself
